# ✨ 4️⃣ INTERACTION SYSTEM

### Apply everywhere:

```css
transition: all 0.2s ease;
```

### Hover behaviors:

* Cards → lift (`shadow-hover`)
* Buttons → slight scale + darker
* Links → underline or color shift

---

# 📱 5️⃣ MOBILE RULES (NON-NEGOTIABLE)

* Use `px-4` always
* Stack everything
* Buttons full width
* Reduce H1 → `text-3xl`

---

# 🧠 6️⃣ DESIGN PRINCIPLES (FOLLOW STRICTLY)

1. **Less clutter = more trust**
2. **Spacing > colors**
3. **One primary action per screen**
4. **Soft UI, not aggressive**
5. **Readable > fancy**
