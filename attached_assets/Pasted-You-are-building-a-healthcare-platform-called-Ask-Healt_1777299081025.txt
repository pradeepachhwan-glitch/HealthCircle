You are building a healthcare platform called "Ask Health".

This is NOT a doctor-only app.
This is NOT a social media app.

This is a COMMUNITY + AI + ACTION platform.

-----------------------------------
🎯 CORE IDEA
-----------------------------------
Users:
- Join health communities
- Ask questions
- Read others’ experiences
- Get AI-assisted summaries
- Take action (consult, tests, etc.)

-----------------------------------
🧱 CORE FEATURES (MANDATORY)
-----------------------------------

1. COMMUNITIES
- Create multiple communities (e.g., Mental Wellness, Diabetes, Pregnancy)
- Each community has:
  - Feed (questions)
  - Threads (answers)
  - Tags

2. THREAD SYSTEM
- Users can:
  - Post questions
  - Reply to others
  - Upvote helpful answers

3. AI LAYER
- Auto-generate for each post:
  - Summary
  - Risk level (low/medium/high)
  - Suggested next steps
- Add “Explain this” button

4. ACTION LAYER
- From each thread:
  - Show doctors
  - Allow booking
  - Suggest tests

5. CHAT ASSISTANT
- Private AI chat for deeper help

-----------------------------------
🧠 AI RULES
-----------------------------------
- No diagnosis
- No prescriptions
- Always structured output:
  1. What this could be
  2. Risk level
  3. What to do
  4. When to see doctor
  5. Disclaimer
